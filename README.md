#  Live Streaming Website Simulation

##  Project Overview

This project simulates a live streaming platform similar to YouTube Live. It includes a video player with autoplay, a live indicator, viewer count simulation, and a live chat interface to mimic a real-time streaming experience.

---

##  Features

* 🔴 LIVE streaming indicator with animation
* 🎥 Autoplay video stream (muted for browser compatibility)
* 👀 Dynamic viewer count simulation
* 💬 Live chat interface (user input + auto messages)
* ⏳ Buffering simulation for realistic experience
* 🌙 Dark theme UI (YouTube-inspired)
* 📱 Responsive design for mobile devices

---

##  Technologies Used

* HTML
* CSS
* JavaScript

---

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. Make sure the video file is placed correctly in the `/video` folder
